基本骨架

直接用 Spring Initializr生成

  • Spring Initializr 界面
  • 项目结构
  • 加点料
package com.wade.demo.web;

import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.RestController;

@RestController
@RequestMapping("/api/v1/hellos")
public class HelloController {

	@GetMapping("")
	public String say() {
		return "Hello World!";
	}
}
  • 运行项目
  • 在浏览器访问http://localhost:8080/api/v1/hellos
  • 修改端口，这里将application改成了.yml格式，看个人习惯，用.properties也可以，只是配置文件里面的格式要注意
server:
  port: 8881
  context-path: /
  • 重新启动，并访问http://localhost:8881/api/v1/hellos
  • 搞定收工！
